My Work

Tuberculosis Detection and Classification Web App

AI-powered web application for tuberculosis detection using MobileNet V3 model. Features image classification and comprehensive summary generation for medical diagnosis. Built with React.js + Tailwind CSS frontend, FastAPI backend, and Python processing.

AI Task Prioritization Tools

Intelligent productivity application that uses LLama LLM to prioritize tasks based on the Eisenhower Matrix. Features smart task ranking and time management insights. Developed with React.js frontend, PostgreSQL database, and Python backend.

Aircraft Damage Detection

Computer vision system using YOLO V5 for automated aircraft damage assessment. Accurately identifies and classifies various types of structural damage. Built with HTML/CSS/JavaScript frontend and Python backend for real-time analysis.

SERGIA: AI Chatbot and Summarization

Advanced AI healthcare assistant using fine-tuned IndoBERT model for patient consultation support. Features intelligent conversation handling, automatic summarization of patient concerns, and pre-consultation report generation for medical professionals. Trained on custom healthcare dataset for improved Indonesian language processing.

Extruder and Mill Installation Project (2025 – ongoing)

Leading electrical and automation scope for new extruder and mill machines installation. Designed and supervised electrical substation with dedicated transformer, power distribution panels, and cabling system. Coordinated end-to-end testing, commissioning, and energization with contractors and maintenance teams for seamless system reliability and compliance.

Digital Architecture Standardization for Calendering Line (2024)

Developed and implemented standardized digital architecture to enhance historical data logging and product traceability for critical calendering machines. Improved data collection and analysis capabilities for better process monitoring and quality control.

Slitter Machine Capacity Improvement & Ergonomic Upgrade (2023)

Increased production capacity from 12,000 to 30,000 units by installing new slitting subassembly machine. Resolved operator ergonomic issues through equipment redesign and workstation optimization, improving both productivity and worker safety.

Calendering Machine High-Risk Safety Project (2023)

Conducted comprehensive risk analysis and implemented advanced safety control circuits through PLC reprogramming and electrical system modifications.
